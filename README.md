# 🌐 Nathaniel Manzano — Developer Portfolio

A personal portfolio website built with **Next.js 14** and **TypeScript**, showcasing my projects, skills, and background as a full stack developer.

## 🚀 Live Demo
[View Portfolio →](https://nathdev.vercel.app/)

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel 

## ✨ Features
- Fully responsive design (mobile-first)
- Server-side rendering with Next.js App Router
- Projects showcase with live links and GitHub links
- Clean, modern UI with smooth navigation

## 📦 Getting Started

```bash
git clone https://github.com/your-username/portfolio
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
